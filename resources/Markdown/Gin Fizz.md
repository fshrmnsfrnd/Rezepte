[[Cocktails]]

>Ein erfrischender Gin Cocktail
# Zutaten
---

| Menge | Zutat         |
| ----- | ------------- |
| 5cl   | Gin           |
| 3cl   | Zitronensaft  |
| 2cl   | Zuckersirup   |
| etwas | Mineralwasser |
# Zubereitung
---
1. Alles außer Wasser shaken
2. In ein Glass und mit Wasser auffüllen
