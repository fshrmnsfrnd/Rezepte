>Wer es Spicy mag
# Zutaten
---

| Menge      | Zutat            |
| ---------- | ---------------- |
| 4cl        | Wodka            |
| 2cl        | Limettensaft     |
| 200ml      | GingerBeer       |
| 5 Scheiben | Gurke (optional) |
| 1 Zweig    | Minze (optional) |
# Zubereitung
---
1. 3 Gurkenscheiben im Shaker zerstoßen
2. Wodka und Limettensaft dazugeben
3. Shaken
4. In ein Glas und mit Gingerbeer auffüllen
