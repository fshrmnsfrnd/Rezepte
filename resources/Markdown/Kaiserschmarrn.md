# Zutaten
---

| Menge   | Zutat                     |
| ------- | ------------------------- |
| 4       | Eier                      |
| 125g    | Mehl                      |
| 125ml   | Milch                     |
| 1 Prise | Salz                      |
| 1 TL    | Backpulver                |
| 60g     | Zucker                    |
| 40g     | Rosinen (optional)        |
| etwas   | Butter                    |
| etwas   | Puderzucker (optional)    |
| etwas   | Apfelmus  (optional)      |
| etwas   | Preiselbeeren  (optional) |
# Zubereitung
---
1. Pfanne erhitzen
2. Eier trennen. 
3. Eigelb mit Mehl, Milch, Salz, Backpulver und Zucker zu einem glatten Teig verarbeiten und 10 Min. ruhen lassen
4. Eiweiß zu einem stabilen Eischnee schlagen und vorsichtig unter den Teig heben. 
5. Anschließend die Rosinen hinzugeben.
6. 2 EL Butter in eine Pfanne geben und bei mittlerer Temperatur einen knusprigen Pfannkuchen braten. 
7. Nach dem backen den Pfannkuchen in Stücke reißen und mit dem Puderzucker bestreuen
8. Tipp: Zerrissene Stücke in einen Topf im Ofen warmhalten und mit Zucker bestreuen
