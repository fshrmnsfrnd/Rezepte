[[Cocktails]]

>Ein Tequila zum Sonnenaufgang?
# Zutaten
---

| Menge | Zutat       |
| ----- | ----------- |
| 4cl   | Tequila     |
| 15cl  | Orangensaft |
| 1cl   | Grenadine   |
# Zubereitung
---
1. Grenadine auf die Eiswürfel im Glas geben
2. Tequila und Orangensaft shaken
3. Vorsichtig ins Glas gießen
